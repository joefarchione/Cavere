# App

Console demo application. References Core and Generators.

## Running

```bash
dotnet run --project src/Cavere.App/Cavere.App.fsproj
```

## Files

- **Stats.fs** — Helper functions: `mean`, `stddev` on float32 arrays
- **Program.fs** — Entry point, builds demo models, runs simulations, prints results

## Namespace

`Cavere.App`
